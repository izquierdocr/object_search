bool ArServerModeDockPatrolBotNiMH::subclassNotDoneCharging ( void  ) [virtual]

This is a call that a subclass can override if it wants to be the only thing that determines done charging, this will inhibit the normal checks of voltage, time, and everything else...

If this returns true then docking will not be done now (note that subclassDoneCharging will not be called if subclassNotDoneCharging returns true)
